# School Management System

> **Note:** This is a repository showcasing my specific contributions to a group project. The complete source code is hosted in the original team repository (linked below).

## 1. Project Overview
A web-based system designed to help university administrators and faculty manage student information, course schedules, and attendance records efficiently.

## 2. My Role & Key Contributions
In this project, I served as a **Full-stack Developer**, responsible for the following modules:

### a. Class Schedule & Attendance Module
* **Front-end:**
    * Developed the user interface for viewing weekly/monthly class schedules for both students and lecturers using HTML, CSS, and JavaScript.
    * Created the attendance tracking interface for lecturers, featuring status options (Present, Absent, Late).
* **Back-end & Database:**
    * Designed related database tables: `Schedules` and `Attendance`.
    * Implemented API/Business logic to retrieve schedule data from the database and render it on the UI.
    * Handled the logic for saving and updating attendance records in the database.

## 3. Technologies Used
* **Front-end:** HTML, CSS, JavaScript.
* **Back-end:** Java.
* **Database:** MySQL.
